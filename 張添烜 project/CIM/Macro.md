# CIM 中的 Macro 概念

## 定義
- **Macro**：在 Compute-In-Memory (CIM) 中，指一個可重複使用的電路模組。
- 封裝了記憶體陣列 + 周邊電路（sense amplifier、ADC/DAC、控制邏輯）。
- 能在記憶體內直接執行計算（如乘加運算）。
## 特點
- **模組化**：像 IP block，一次設計，多次例化。
- **運算與儲存結合**：減少記憶體牆問題。
- **設計層次**：介於電路級與系統級之間，方便面積/功耗/效能估算。
## 舉例
- **SRAM-based CIM Macro**：支援矩陣乘法、CNN 推論。
- **9T SRAM CIM Macro**：利用電荷再分配做類比計算，降低能耗。

- **像 Verilog 的 module**：封裝功能，設計者直接例化使用。
